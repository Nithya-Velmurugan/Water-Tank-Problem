# Water Tank Problem - Visualizer

An interactive Frontend Web Application built with Vanilla JavaScript, HTML, and CSS to visualize and calculate the generic Water Tank Problem (also known as Trapping Rain Water).

## Project Overview
Given an array `n` representing block heights, this application calculates the precise units of water stored horizontally in-between the blocks after a rain. 

The visualizer provides a dynamic 1-to-1 Table View matching the standard problem description.

## Instructions
1. Open `index.html` in any modern web browser.
2. Enter a comma-separated list of integers representing your tank block heights (e.g., `0,4,0,0,0,6,0,6,4,0`).
3. Click **Compute** to instantly view the calculated array units alongside the graphical SVG table projections.

## Technologies Used
- HTML5
- CSS3 (Modern Flexbox + Inter Font)
- Vanilla JavaScript (Zero external dependencies)
